Comandos

Veja os comandos detalhados em ordem de lançamento, lembre-se, sempre estamos adicionando novos comandos e melhorando o bot.

  • @(nomeDoBot)

    • Quando mençionado, o bot retorna uma mensagem

  • Help

    • Após o comando, o bot retorna uma mensagem de ajuda para o usuário

  • Ping

    • Após o comando, o bot retorna uma mensagem que mostra a latência

  • Invite

    • Após o comando, o bot retorna uma mensagem que envia um link de convite para o bot entrar no servidor do usuário

  • Server

    • Após o comando, o bot retorna uma mensagem que mostra dados sobre o servidor atual

  • Say

    • Após o comando, o bot retorna uma mensagem igual a do usuário

  • Daily

    • Após o comando, o bot retorna uma mensagem que revela quanto o jogador ganhou no prêmio diario

  • Balance
